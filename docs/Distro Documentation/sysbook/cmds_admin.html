Sysbook package Documentation - Admin Commands

For the sysbook package in the Distro, by Yukiko.

Administrator Commands

This is a list of the administrator (Admin) level commands and their description:
addtomasterlibrary
 Used to add a new book to the Master Library (sysbook:master_library.cfg). After entering the command the user is promted to target a book. The book is checked to be sure it has content and if so, the contents are reformatted to match the books config file format. Then the book info is appended to the end of the Master Library (:sysbook:master_library.cfg). When the user is done, press <ESC> to exit.
addtoshardlibrary
 This command will add a book from the Master Library (:sysbook:master_library.cfg) to the Shard Library (shard_library data file).  This will make that book available for creation in-game and, if the book is set to spawn when it is selected, it will spawn in bookshelves that are set to spawn books.
createshardlibrary
 This command creates all of the Shard Library's books in a container of your choice. Please ensure the container can hold a lot of items.
removefromshardlibrary
 This command will remove selected books from the Shard Library. The book will still be available in the Master Library and can be added back into the Shard Library by using the addtoshardlibrary command.
setbooktospawn
 This allows the user to set certain books in the Shard Library to spawn in the bookshelves that are set to spawn books.
setbooknottospawn
 This command allows the user to stop certain books in the Shard Library from spawned in the bookshelves that are set to spawn books
